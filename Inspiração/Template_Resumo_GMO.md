# Template de Resumo GMO

Este arquivo serve como base para preencher os dados do documento Word.
As variáveis abaixo correspondem aos textos exatos que estão no `Template Word.docx` e serão substituídos pelo gerador.

## Variáveis da Capa e Controle de Versão

- **[title]**: Resumo de Atividades GMO
- **[Subtitle]**: Gerenciamento de Mudança Organizacional (SAP S/4HANA)
- **[[Publish Date]]**: 04/03/2026
- **[[Version]]**: 1.0
- **[(e.g. name customer)]**: Cliente / Projeto SAP S/4HANA
- **[(name contact person)]**: Consultor Especialista GMO

## Conteúdo do Resumo

O conteúdo principal (Estratégia, Mapeamento Integrado, Treinamento, etc) será adicionado automaticamente ao final do documento pelo script `gerar_gmo.py`.
