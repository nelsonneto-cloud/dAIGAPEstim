# Template de Especificação Funcional

Este arquivo serve como base para preencher os dados do documento Word.
As variáveis abaixo correspondem aos textos exatos que estão no `Template Word.docx` e serão substituídos pelo gerador.

## Variáveis da Capa e Controle de Versão

- **[title]**: Especificação Funcional - Espelho de Pedido de Compras SAP S/4HANA
- **[Subtitle]**: Relatório de Compras Consolidado
- **[[Publish Date]]**: 04/03/2026
- **[[Version]]**: 1.0
- **[(e.g. name customer)]**: Cliente / Projeto SAP S/4HANA
- **[(name contact person)]**: Consultor Responsável

## Conteúdo da Especificação

O conteúdo principal (Objetivo, Regras de Negócio, etc) será adicionado automaticamente ao final do documento pelo script `gerar_ef.py`.
